### Third：归一化数值
> - 在处理这种不同取值范围的特征值时，我们通常采用的方法是将数值归一化，如将取值范围
> 处理为 0 到 1 或者 -1 到 1 之间。处理归一化的时候有公式：
> newValue = (oldValue - min) / (max - min)
> 其中 min 和 max 分别表示最小特征值和最大特征值
